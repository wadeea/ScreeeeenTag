# ESL Platform Design Preferences

## Aesthetic: "Industrial Tech/Command Center"
- **Primary Color**: Zinc-950 (Background), Emerald-500 (Status/Success), Amber-500 (Pending/Task).
- **Typography**: Inter (UI), JetBrains Mono (Tag IDs/Logs).
- **Cards**: Flat, subtle borders (zinc-800), high density.

## Components
- Real-time pulse indicators for AP status.
- Battery level icons for tags.
- Task queue progression sidebar.
